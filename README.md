# AI FAQ Chatbot 🤖

## 📌 Description
An AI-powered FAQ chatbot built using Python, Streamlit, Pandas, NLTK, and Scikit-learn. It answers frequently asked questions using Natural Language Processing (NLP).

## 🚀 Features
- AI-based FAQ answering
- NLP text preprocessing
- TF-IDF Vectorization
- Cosine Similarity matching
- Chat history
- Simple Streamlit interface

## 🛠️ Technologies Used
- Python
- Streamlit
- Pandas
- NLTK
- Scikit-learn

## ▶️ Run the Project

```bash
pip install -r requirements.txt
streamlit run app.py
```
